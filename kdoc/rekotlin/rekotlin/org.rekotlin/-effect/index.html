interface Effect : Dispatchable

Marker interface for effects dispatched to a Store.

Effects are ephemeral, they do not change the store's state, however listeners can react to them. Effects are a concept with ephemeral application behavior in mind, such as triggering an animation, showing a toast or showing a snackbar.

All effects need to be marked with this interface.
