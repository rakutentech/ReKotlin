Middleware
Link copied to clipboard
typealias Middleware<State> = (DispatchFunction, () -> State?) -> (DispatchFunction) -> DispatchFunction
Content copied to clipboard

A middleware intercepts all actions dispatched to the store before they reach the reducers.

Middlewares have access to the previous state, can drop actions or dispatch others (potentially multiple others. This makes middlewares very powerful. Any middleware will affect all actions dispatched to the store, so use them for cross-cutting concerns only.
